This project analyzes the impact of mobile and social media overuse on mental health factors such as stress, anxiety, and sleep quality. The study is based on survey data and uses statistical methods and visualizations to understand usage patterns and their relationship with mental well-being.

--Project Overview

The objective of this project is to study how excessive mobile usage affects individuals across various age groups and user categories.
The analysis covers:

Stress levels

Anxiety indicators

Sleep disturbances

Duration and frequency of mobile/social media use

Comparison across different user groups

 --Methods & Statistical Analysis

--Survey Data Collection:
Collected responses through a structured questionnaire to measure mobile usage habits and psychological effects.

--Statistical Tests Used:

Chi-Square Test – To examine associations between categorical variables

t-Test – To compare mean differences between user groups

--Tools Used:

Python (Pandas, NumPy, SciPy, Matplotlib)

Excel for initial cleaning and validation

--Visualizations

Created meaningful and easy-to-interpret charts, including:

Bar Charts

Pie Charts

These visuals highlight usage patterns and mental health variations across different groups.

--Key Findings

Found a significant correlation between mobile addiction and poor mental health outcomes.

Participants with higher screen time experienced increased stress, anxiety, and sleep difficulties.
